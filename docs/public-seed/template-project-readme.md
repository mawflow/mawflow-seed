# 业务项目 README 模板

> 使用 Mawflow Seed 创建新项目后，可以把本文件内容复制到根目录 `README.md`，再按当前项目事实填写。

## 项目简介

- 项目名称：填写当前项目名称
- 项目负责人：填写负责人或团队
- 业务简介：用一两句话说明项目目标
- 当前状态：填写开发、测试、上线或维护状态

## 快速开始

```bash
# 填写安装依赖、启动服务、运行测试或构建命令
```

## 目录入口

- 模板使用说明：`TEMPLATE_OVERVIEW.md`
- 新手指南：`GETTING_STARTED.md`
- 人类常用指令目录：`PROJECT_COMMANDS.md`
- ChatGPT 到 Codex 任务交接协议：`CHATGPT_TO_CODEX.md`
- AI 协作上下文：`.maw/codex-context.md`
- 项目文档入口：`docs/README.md`
- 代码目录入口：`code/README.md`

## AI 协作提示

- `.maw/` 保存 AI/人工协作用配置索引。
- `docs/ai-coding/` 保存 AI 编码边界和组件说明。
- `docs/ai-instructions/` 保存项目内指令、术语和经验索引。
- `docs/modules/` 和 `.maw/modules.yaml` 保存模块边界。

模板升级或模板化改造时，不应整文件覆盖目标项目 README；如需补充模板入口，只做最小段落合并。
